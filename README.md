# nosql2h22-rospatent

## Установка зависимостей и запуск

```bash setup.sh```

```source env/bin/activate```

```cd rospatent```

```python3 manage.py runserver 8081```

Можно открыть страницу по адресу ```http://127.0.0.1:8081/```.
